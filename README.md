# Sales-Performance-Dashboard


## Overview
This project is an interactive Power BI dashboard designed to analyze sales performance.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Category Analysis
- Sales by Region Analysis
- Monthly Sales Trend
- Interactive Filters (Region & Category)

## Tools Used
- Power BI
- Data Visualization
- DAX
